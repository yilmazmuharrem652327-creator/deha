Birinci Büyü

Console.WriteLine — Söz Söyleme Büyüsü

Bu, öğreneceğin en güçlü ilk büyü. Console.WriteLine dediğinde, bilgisayarın ekranına istediğin yazıyı gösterir. Parantez içine yazdığın her şey, tırnak işaretleri arasında bir söz olarak ortaya çıkar.

Buyu.cs
using System;

Console.WriteLine("Merhaba, ben Deha!");
Console.WriteLine("C# ile büyü yapıyorum.");
Ekranda görünen Merhaba, ben Deha!
C# ile büyü yapıyorum.
🧪
İkinci Büyü

string ve int — Büyü Malzemeleri

Her büyücünün bir malzeme çantası vardır. C#'ta da bilgiyi saklamak için değişkenler kullanırız. İki temel malzeme türü öğreneceğiz: string (kelime/metin) ve int (tam sayı).

Malzemeler.cs
string isim = "Deha";
int yas = 10;

Console.WriteLine(isim);
Console.WriteLine(yas);
🔤

string

Kelime ve cümleleri saklar. Her zaman "çift tırnak" içinde yazılır. Örnek: "Deha"

🔢

int

Tam sayıları saklar. Tırnak kullanılmaz. Örnek: 10, 2026

💫
Üçüncü Büyü

$" {değişken} " — Birleştirme Büyüsü

İşte akademinin en gösterişli büyüsü! Yazının başına $ işareti koyup, süslü parantez { } içine bir değişken adı yazarsan, o değişkenin değeri doğrudan cümlenin içine karışır. Buna string interpolation denir.

BirlestirmeBuyusu.cs
string isim = "Deha";
int yas = 10;

Console.WriteLine($"Merhaba {isim}, sen {yas} yaşındasın!");
Ekranda görünen Merhaba Deha, sen 10 yaşındasın!
🏆 3 büyüyü de öğrendin, Deha!
